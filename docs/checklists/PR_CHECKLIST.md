# PR Checklist

Before merge:

- [ ] Scope is focused (1–3 concerns per PR).
- [ ] Contract impact stated (`none` or list under `docs/contracts/`).
- [ ] Tests added/updated for behavior changes; `npm test` passes.
- [ ] `npm run build` passes.
- [ ] Manual XR/desktop checks done where relevant.
- [ ] No obvious perf regressions.
